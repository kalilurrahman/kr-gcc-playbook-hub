1. Zinnov Four-Wave Maturity Framework

Wave 1: Outpost (~30%)

Remote operational arm; HQ dictates all; task execution only.

Wave 2: Satellite (~35%)

Process ownership, standardization, cost efficiency.

Wave 3: Portfolio Hub (~27%)

CoEs, domain expertise, global influence in 2–3 areas.

Wave 4: Transformation Hub (8–19%)

AI-native, global mandates, IP generation, enterprise co-leadership.
